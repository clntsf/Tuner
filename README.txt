# Tuner
Sinewave Tuner App for SPL

Installation command (copy and paste into terminal):
curl https://github.com/ctsf1/Tuner/archive/main.zip -L -O; unzip -o tuner_resources.zip; python3 tuner_resources/install_tuner.py
